screen 使用记录

发表于2021-11-05|更新于2021-11-05
|阅读量:

screen

创建会话：screen -S name

退出会话：ctrl A + D

恢复会话：screen -x

显示目前所有的screen作业：screen -ls或–list

完全退出：exit
